## 📌 Project Overview
This is a full-stack Todo application built using React (frontend) and Node.js + Express (backend).  
It allows users to create, view, update, delete, and search tasks with a neon-themed UI.

---

## 🚀 Features

### Frontend Features
- Multi-page React application (not SPA)
- Todo List Page
- Todo Detail Page (query parameter based routing)
- Add new tasks with title and description
- Update task status (Pending / Completed)
- Delete tasks instantly
- Search tasks
- Live task counters (Total & Completed)
- Neon animated UI with smooth transitions

---

### Backend Features
- Node.js + Express server
- REST API (CRUD operations)
- File-based storage using db.json
- Endpoints:
  - GET /todos
  - GET /todos/:id
  - POST /todos
  - PUT /todos/:id
  - DELETE /todos/:id
